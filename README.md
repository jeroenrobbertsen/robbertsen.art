# Robbertsen Art — Portfolio Website

## Setup

### 1. Voeg je afbeeldingen toe

Plaats de volgende bestanden in de `public/images/` map:

| Bestandsnaam            | Beschrijving         |
|-------------------------|----------------------|
| `logo-white.png`        | Je witte logo        |
| `fire-water-earth.jpg`  | Fire Water Earth     |
| `creatures-02.jpg`      | Creatures 02         |
| `terschelling.jpg`      | Terschelling         |
| `figures-01.jpg`        | Figures 01           |
| `figures-02.jpg`        | Figures 02           |
| `forms-00.jpg`          | Forms 00             |
| `forms-01.jpg`          | Forms 01             |

Ideale breedte: **1200px** (JPG, kwaliteit 80-85%)

### 2. Lokaal testen

```bash
npm install
npm run dev
```

### 3. Deployen

Push naar GitHub → koppel aan Vercel → klaar.
